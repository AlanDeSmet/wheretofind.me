============================================
Welcome to Where to Find Me's documentation!
============================================

Where to Find Me helps people find each other as social media sites come
and go. It's pretty much a good-old-fashioned server-side-rendered
Django app, but it has a little fancy interactivity from Vue attached to
the frontend.
